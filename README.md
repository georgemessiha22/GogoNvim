## Install Instructions

 > Install requires Neovim 0.12+. Always review the code before installing a configuration.

In your Neovim init config

```lua
vim.pack.add({ { src = "https://github.com/georgemessiha22/GogoNvim", version = "feat/convert_to_pack", data = { confirm = false } } })

require("gogovim").setup()
```
